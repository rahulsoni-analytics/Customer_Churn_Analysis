# Customer_Churn_Analysis
This project focuses on analyzing customer churn behavior using a telecom customer dataset to identify the key factors that influence customer attrition. The analysis aims to uncover actionable insights that can help businesses design effective customer retention strategies.
The dataset contains 7,043 customer records and 21 features, including demographic details, service usage, contract type, payment method, and billing information. Python-based Exploratory Data Analysis (EDA) techniques were used to understand patterns and trends related to churn.
